
#################### USAGE INSTRUCTIONS ####################

Run the Sensor_Dashboard_V2_Deployed.py file to start the sensor dashboard.


The CO2, Ozone, Temp, Pressure buttons all toggle the respective graphs.


The Notif. Button toggles the Danger Notification. 
The entire screen will change colour based on the folowing gas levels:

	CO2	Ozone	Level
White	<1000	<0.1	Safe
Yellow	1000	0.1	Above 8 hour working limit
Red	5000	0.2	Dangerous

These have been deemed to be appropriate levels based on online sources.
When the mouse is moved away from the Notif button it will show its status.
Green = Active, showing danger notifications
Red = Not active, not showing danger notifications


The Quit button will exit the program.
